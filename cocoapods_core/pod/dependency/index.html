Dependency

The Dependency allows to specify dependencies of a Podfile or a Specification on a Pod. It stores the name of the dependency, version requirements and external sources information.

This class leverages the RubyGems dependency class with minor extension to support CocoaPods specific features.

More information can be found at:

  • https://github.com/rubygems/rubygems/blob/master/lib/rubygems/dependency.rb
  • http://rubygems.rubyforge.org/rubygems-update/Gem/Dependency.html

The Dependency class provides support for subspecs.

282 def external_source_description
283   source = external_source
284   if source.key?(:git)
285     desc =  "`#{source[:git]}'"
286     desc << ", commit `#{source[:commit]}'" if source[:commit]
287     desc << ", branch `#{source[:branch]}'" if source[:branch]
288     desc << ", tag `#{source[:tag]}'"       if source[:tag]
289   elsif source.key?(:podspec)
290     desc = "`#{source[:podspec]}'"
291   elsif source.key?(:local)
292     desc = "`#{source[:local]}'"
293   else
294     desc = "`#{source.to_s}'"
295   end
296   "from #{desc}"
