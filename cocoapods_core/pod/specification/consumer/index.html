Consumer

Allows to conveniently access a Specification programmatically.

It takes care of:

  • standardizing the attributes
  • handling multi-platform values
  • handle default values
  • handle inherited values

This class allows to store the values of the attributes in the Specification as specified in the DSL. The benefits is reduced reliance on meta programming to access the attributes and the possibility of serializing a specification back exactly as defined in a file.

237 def merge_values(attr, existing_value, new_value)
238   return existing_value if new_value.nil?
239   return new_value if existing_value.nil?
240 
241   if attr.types.include?(TrueClass)
242     new_value.nil? ? existing_value : new_value
243   elsif attr.container == Array
244     r = [*existing_value] + [*new_value]
245     r.compact
246   elsif attr.container == Hash
247     existing_value = existing_value.merge(new_value) do |_, old, new|
248       if new.is_a?(Array) || old.is_a?(Array)
249         r = [*old] + [*new]
250         r.compact
251       else
252         old + ' ' + new
253       end
254     end
255   else
256     new_value
257   end
258 end
