Version

The Version class stores information about the version of a Specification . It is based on the RubyGems class (http://rubygems.rubyforge.org/rubygems-update/Gem/Version.html) with minor adaptations for CocoaPods.

def self.from_string(string)
  if string =~ /HEAD (based on|from) (.*)/
    v = Version.new($2)
    v.head = true
    v
  else
    Version.new(string)
  end
end
