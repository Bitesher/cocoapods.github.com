==

LocalSource

Provides support for fetching a specification file from a path local to the machine running the installation.

Works with the LocalPod::LocalSourcedPod class.

Inherits from:

130 def pre_download(sandbox)

132     target = sandbox.root + name
133     target.rmtree if target.exist?

318 def pod_spec_path
319   declared_path = params[:local].to_s
320   path_with_ext = File.extname(declared_path) == '.podspec' ? declared_path : "#{declared_path}/#{name}.podspec"
321   podfile_dir   = File.dirname(podfile_path || '')
322   absolute_path = File.expand_path(path_with_ext, podfile_dir)
323   pathname      = Pathname.new(absolute_path)
324

327   end
