AdvancedLinter

Extends the Linter from the Core to add additional which require the LocalPod and the Installer.

In detail it checks that the file patterns defined by the user match actually do match at least a file and that the Pod builds, by installing it without integration and building the project with xcodebuild.

253 def parse_xcodebuild_output(output)
254   lines = output.split("\n")
255   selected_lines = lines.select do |l|
256     l.include?('error: ') &&
257       (l !~ /errors? generated\./) && (l !~ /error: \(null\)/)  ||
258       l.include?('warning: ') && (l !~ /warnings? generated\./) ||
259       l.include?('note: ') && (l !~ /expanded from macro/)
260   end
261   selected_lines.map do |l|
262     new = l.gsub(/\/tmp\/CocoaPods\/Lint\/Pods\//,'') # Remove the unnecessary tmp path
263     new.gsub!(/^ */,' ')                              # Remove indentation
264     "XCODEBUILD > " << new                            # Mark
265   end
266 end
