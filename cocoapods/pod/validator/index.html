Validator

Validates a Specification.

Extends the Linter from the Core to add additional which require the LocalPod and the Installer.

In detail it checks that the file patterns defined by the user match actually do match at least a file and that the Pod builds, by installing it without integration and building the project with xcodebuild.

327 def parse_xcodebuild_output(output)
328   lines = output.split("\n")
329   selected_lines = lines.select do |l|
330     l.include?('error: ') &&
331       (l !~ /errors? generated\./) && (l !~ /error: \(null\)/)  ||
332       l.include?('warning: ') && (l !~ /warnings? generated\./) ||
333       l.include?('note: ') && (l !~ /expanded from macro/)
334   end
335   selected_lines.map do |l|
336     new = l.gsub(/\/tmp\/CocoaPods\/Lint\/Pods\//,'')
337     new.gsub!(/^ */,' ')
338   end
339 end
