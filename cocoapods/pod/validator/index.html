Validator

Validates a Specification.

Extends the Linter from the Core to add additional which require the LocalPod and the Installer.

In detail it checks that the file patterns defined by the user match actually do match at least a file and that the Pod builds, by installing it without integration and building the project with xcodebuild.

345 def parse_xcodebuild_output(output)
346   lines = output.split("\n")
347   selected_lines = lines.select do |l|
348     l.include?('error: ') &&
349       (l !~ /errors? generated\./) && (l !~ /error: \(null\)/)  ||
350       l.include?('warning: ') && (l !~ /warnings? generated\./) ||
351       l.include?('note: ') && (l !~ /expanded from macro/)
352   end
353   selected_lines.map do |l|
354     new = l.gsub(/\/tmp\/CocoaPods\/Lint\/Pods\//,'')
355     new.gsub!(/^ */,' ')
356   end
357 end
