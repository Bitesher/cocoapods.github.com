container_portal

Signature

- (String) container_portal
TODO

This is an attribute because a it is usually a reference to the root object or to a file reference to another project. The reference to the root object causes a retain cycle that could cause issues (e.g. in totreehash). Usually those objects are retained by at least another object (the {Project} for the root object and a {PBXGroup} for the reference to another project) and so the referenced object should be serialized.

If this assumption is incorrect, there could be loss of information opening and saving an existing project.

Returns

String

Apparently the UUID of the root object {PBXProject} of the project containing the represented object.

def remove_from_project
  @project.objects_by_uuid.delete(uuid)
  @referrers.each { |referrer| referrer.remove_reference(self) }
  raise "[Xcodeproj] BUG: #{self} should have no referrers instead the following objects are still referencing it #{referrers}" unless referrers.count == 0
end

def to_plist
  plist = {}
  plist['isa'] = isa

  simple_attributes.each do |attrb|
    value = attrb.get_value(self)
    plist[attrb.plist_name] = value if value
  end

  to_one_attributes.each do |attrb|
    obj = attrb.get_value(self)
    plist[attrb.plist_name] = obj.uuid if obj
  end

  to_many_attributes.each do |attrb|
  list = attrb.get_value(self)
    plist[attrb.plist_name] = list.uuids
  end

  references_by_keys_attributes.each do |attrb|
    list = attrb.get_value(self)
    plist[attrb.plist_name] = list.map { |dictionary| dictionary.to_plist }
  end

  plist
end
